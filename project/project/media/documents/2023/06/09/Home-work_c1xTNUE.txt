
Доработайте ваше приложение так, чтобы оно реализовало все принципы REST API:

1. Для новостей сделать возможность получить их по пути /news.
2. Аналогично статьи находятся по пути /articles.
3. Авторизованные пользователи имеют возможность отослать все виды запросов (POST, PUT, DELETE).
4. Неавторизированные — только читать (GET, HEAD, OPTIONS).

-----------------------------------------------------------------------------------------------------------

Реализовать поддержку перевода новостей на несколько языков:

1. Добавить возможность выбора языка для пользователя (хватит русского и английского).
2. Если пользователь выбрал язык — обновляем страницу и показываем новое содержимое.
3. При написании статьи есть возможность из админки написать несколько вариантов перевода.
4. Переводить достаточно содержимое и заголовок. Содержание можно оставить, как есть.
